## 📋 Настройка GitHub Pages

### Шаг 1: Загрузите файлы в репозиторий

```bash
# Перейдите в папку presentation
cd presentation

# Инициализируйте git (если ещё не сделано)
git init

# Добавьте удалённый репозиторий
git remote add origin https://github.com/richboy13/bridge_presentation.git

# Добавьте все файлы
git add .

# Создайте коммит
git commit -m "Добавлены презентации Bridge"

# Загрузите в GitHub
git branch -M main
git push -u origin main
```

### Шаг 2: Включите GitHub Pages

1. Перейдите в репозиторий на GitHub: https://github.com/richboy13/bridge_presentation
2. Нажмите **Settings** (Настройки)
3. В левом меню найдите **Pages**
4. В разделе **Source** выберите:
   - Branch: `main`
   - Folder: `/ (root)` или `/presentation` (если файлы в подпапке)
5. Нажмите **Save**
6. Через 1-2 минуты ваши презентации будут доступны по адресу:
   **https://richboy13.github.io/bridge_presentation/**

### Шаг 3: Настройте главную страницу

Если хотите, чтобы `hub.html` открывался по умолчанию, переименуйте его в `index.html`, а текущий `index.html` переименуйте в `investor.html`.

## Использование

1. Откройте файл `index.html` в любом современном браузере
2. Используйте кнопки "Назад" и "Вперёд" для навигации
3. Или используйте стрелки на клавиатуре (← →)
4. Для полноэкранного режима нажмите F11

4. **Решение — Bridge** - Описание платформы
5. **Архитектура платформы** - Обзор модулей
6. **Казахстанская платформа (B2B)** - Функционал для юр. лиц РК
7. **Китайская платформа** - Работа с производителями
8. **Операционный модуль КНР** - Расчёт логистики и цен
9. **Операционный модуль РК** - Расчёт таможни и финальная цена
10. **Цепочка исполнения заказа** - Процесс от заказа до доставки
11. **Административный модуль** - Управление процессами
12. **Экосистема внешних операторов** - Диверсификация доходов
13. **Ценность для клиента** - Преимущества платформы
14. **Бизнес-модель** - Источники дохода
15. **Дорожная карта** - Фазы развития
16. **Инвестиционное предложение** - Условия инвестирования
17. **Финал** - Итоговые выводы
